# Checklist de ejercicio

## Preparacion

- [ ] Rama creada desde el ejercicio anterior.
- [ ] `.env` revisado.
- [ ] Dependencias instaladas.
- [ ] Base de datos regenerada antes de empezar.

## Desarrollo

- [ ] Migraciones creadas o revisadas.
- [ ] Modelos y relaciones actualizados.
- [ ] Seeders o factories preparados.
- [ ] Rutas web/API registradas.
- [ ] Controladores implementados.
- [ ] Resources creados si el ejercicio devuelve API.
- [ ] Middleware o Policies aplicados si el ejercicio lo pide.
- [ ] Validaciones incluidas.
- [ ] Correos o ficheros comprobados si aparecen en el enunciado.

## Comprobacion

- [ ] `php artisan migrate:fresh --seed` funciona.
- [ ] `php artisan route:list` muestra las rutas esperadas.
- [ ] Navegador o cliente API probado manualmente.
- [ ] Codigos HTTP comprobados en errores.
- [ ] Commit realizado.
